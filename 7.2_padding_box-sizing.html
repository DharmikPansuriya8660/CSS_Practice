Padding and element width - with box-sizing

This div is 300px wide.

box-sizing : border-box = This will set the text to suit the width and padding.

👉If this is not the case then the size of the whole box will increase due to the width .... which should not be done ... box sizing is necessary if the size is to be kept the same despite giving cushion.

The width of this div remains at 300px, in spite of the 50px of total left and right padding, because of the box-sizing: border-box property.
